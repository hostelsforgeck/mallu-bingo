how-to-play

this is a 2-Player Game

  1. Each player has a 5x5 board with numbers 1-25, randomly placed
  2. Keep your board hidden during gameplay
  3. Player1 picks an unmarked number - both players mark it
  4. Player2 picks an unmarked number - both players mark it
  5. Get points by completing rows, columns, or diagonals
  6. First to score 5 points (B I N G O) wins!
